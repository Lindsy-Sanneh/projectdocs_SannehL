Lost in Tra(i)nslation

Week 6 Proces

In de zesde week zijn we verdergegaan met de code, maar de belangrijkste stap was het leren gebruiken van GitHub en Netlify. Ik heb geleerd hoe ik mijn bestanden kan uploaden naar GitHub om ze veilig te bewaren. Daarna heb ik mijn project gekoppeld aan Netlify, zodat mijn website ook echt live op internet te bezoeken is via een link. Het was spannend om te zien dat de code die ik op mijn computer schreef, nu ook echt voor iedereen online staat.
